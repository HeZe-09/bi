# 智能BI平台
  区别于传统的BI，用户只需要导入原始数据集，并输入分析诉求，就能自动生成可视化图表及分析结论，实现数据分析的降本增效。

技术栈：

  前端
  1. React 18
  2. Ant Design Pro 框架
  3. Umi4前端框架
  4. Ant Design组件库
  5. Echarts可视化库
  6. OpenAPI前端代码生成


后端
1. Java Spring Boot
2. MySQL数据库
3. MyBatis-Plus 及 MyBatis X自动生成
4. Redis + Redisson 限流
5. RabbitMQ 消息队列
6. Open AI SDK
7. Easy Excel 表格数据处理
8. Hutool等工具库
